## Idea 1: Neo-Brutalist Dark Craft

**Design Movement**: Neo-Brutalism meets Dark Editorial

**Core Principles**:
- Raw typographic power — oversized display type as structural element
- High-contrast dark canvas with warm amber/orange accent glows
- Asymmetric card-based layout with deliberate tension
- Monospace + script font collision for personality

**Color Philosophy**:
- Background: near-black #0d0d0d with subtle warm undertone
- Cards: #1a1a1a with slight warm tint

- Text: off-white (#f0ede8) for warmth, not clinical white
- Muted: #666 for secondary text

- Sticky left sidebar card (profile) with scrolling right content
- Sections stacked vertically with generous breathing room
- Stats cards in a 2-column grid with large counter typography
- Portfolio items as full-width stacked cards

- Glowing orange halo behind profile photo
- Marquee/ticker text for "Selected Work" section
- Script font "Jayden" signature overlay on photo

**Interaction Philosophy**:
- Hover states reveal arrow-link buttons
- Cards lift slightly on hover with subtle glow
- Smooth scroll with section fade-in animations

**Animation**:
- Counter animations for stats (10M+, 10%)
- Marquee scrolling text ticker
- Fade-up entrance for each section
- Hover: card scale(1.01) + glow intensify

**Typography System**:
- Display: Clash Display or Space Grotesk (bold, geometric)
- Script: Dancing Script for "Jayden" signature
- Body: DM Sans (clean, readable)
- Monospace accents: JetBrains Mono for labels/tags

## Idea 3: Dark Organic Warmth (CHOSEN)

**Design Movement**: Dark Organic Minimalism — warm blacks, glowing amber, editorial structure

**Core Principles**:
- Warmth in darkness: near-black backgrounds with amber/orange glow accents
- Typography as architecture: large, confident display text anchors each section
- Floating card sidebar: profile card stays fixed while content scrolls
- Deliberate restraint: sparse UI, maximum impact per element

**Color Philosophy**:
- Background: #0e0e0e (warm near-black, not cold gray)
- Card surfaces: #181818 / #1f1f1f
- Primary accent: #ff6b2b (warm orange, matching the photo's ambient glow)
- Secondary accent: #ff4500 dot indicator
- Text primary: #f5f0ea (warm off-white)
- Text muted: #888 / #666
- Border: rgba(255,255,255,0.08)

**Layout Paradigm**:
- Left-fixed profile card (~280px) + right scrollable content
- Content sections flow vertically with large section labels
- Stats in 2-up dark cards with massive counter type
- Experience as clean list with date badges
- Portfolio as stacked full-width cards with image backgrounds

**Signature Elements**:
- Glowing amber halo behind profile portrait (radial gradient)
- Script font signature "Jayden" overlaid on photo
- Orange dot availability indicator
- Circular arrow-button CTAs

**Interaction Philosophy**:
- Hover reveals arrow-link on portfolio cards
- Service accordion expands sub-items
- Smooth scroll-triggered fade-up for sections

**Animation**:
- CountUp animation for stats (10M+, 10%)
- Marquee ticker for "Selected Work" header
- Fade-up + slight translateY for section entrances
- Hover: subtle scale + glow on cards

**Typography System**:
- Display: Clash Display / Space Grotesk ExtraBold for headings
- Script: Dancing Script for "Jayden" signature
- Body: DM Sans Regular/Medium
- Labels: uppercase tracking-widest for section labels
